# 2 补充
## 2.1 代码高亮
### 使用三个反引号 ``` 包裹代码块，并在起始的三个反引号后指定代码的语言。以下是一些常见的代码高亮语言标识符的示例：
    cpp 或 c++：C++代码
    python：Python代码
    java：Java代码
    javascript：JavaScript代码
    html：HTML代码
    css：CSS代码
    bash 或 shell：Shell脚本代码
### 例如：
```c++
#include <iostream>

int main() {
    std::cout << "Hello, world!" << std::endl;
    return 0;
}
```
